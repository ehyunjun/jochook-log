# 조축로그

조축로그는 조기축구 팀을 위한 쿼터별 포메이션 작성 및 경기 기록 웹입니다.

팀장이 경기 당일 빠르게 팀원을 등록하고, 쿼터별 포메이션을 만든 뒤, 경기 후 득점/어시스트/출전횟수 같은 기록을 간단히 남길 수 있도록 만든 가벼운 MVP입니다.

## 주요 기능

- 팀 생성 및 임시 팀 코드 표시
- 서버 검증 없는 팀 코드 참가 흐름
- 팀원 등록, 삭제, 일일 용병 자동 추가
- 팀원별 출전횟수, 득점, 어시스트 누적 기록 확인
- 팀원 카드 클릭 시 상세 기록 및 최근 경기 기록 확인
- 1~4쿼터 설정 및 쿼터별 포메이션 저장
- 4-4-2, 4-3-3, 4-2-3-1 포메이션 선택
- 축구장 형태 스쿼드판에서 포지션별 선수 선택
- 포지션별 메모 저장
- 경기명, 날짜, 상대팀/메모, 출전 선수, 득점, 어시스트 기록
- 저장된 경기 기록 목록 표시
- 현재 쿼터 스쿼드 공유용 텍스트 복사

## 사용 기술

- HTML
- CSS
- JavaScript
- localStorage

별도 백엔드, DB, 외부 라이브러리 없이 동작합니다.

## 실행 방법

1. 프로젝트 폴더에서 `index.html` 파일을 브라우저로 엽니다.
2. 팀을 생성하거나 팀 코드로 참가합니다.
3. 팀원을 추가하고 포메이션과 경기 기록을 작성합니다.

저장 데이터는 브라우저 `localStorage`에 보관됩니다.

## 추후 추가 예정 기능

- 실제 팀 초대/참가 기능
- 이미지 캡처 저장 기능
- 경기별 상세 리포트 기능
- 팀원별 누적 기록판 고도화
- 로그인 및 관리자 권한 기능
